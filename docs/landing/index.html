LicenseDesk

Open-source система учёта нематериальных активов: лицензий, подписок, сертификатов, доменов

GitHub Быстрый старт

Возможности

📦

Учёт активов

Лицензии, подписки, домены, SSL-сертификаты, облачные сервисы. Привязка к организациям и проектам. Архивирование с возможностью восстановления.

💰

Платежи и стоимость

История платежей, автоматический расчёт следующей оплаты. Фиксированный и плавающий периоды продления. Прикрепление инвойсов.

🔔

Уведомления

Настраиваемые напоминания о платежах. Индивидуальные настройки для каждого актива. Возможность отключения.

🔐

Контроль доступа

Три роли: Администратор, Менеджер, Наблюдатель. Права привязаны к организациям. Фильтрация данных по доступу.

📊

Экспорт и аудит

Выгрузка в Excel с фильтрами. Полный журнал действий с детализацией изменений.

🔌

Интеграции

Keycloak SSO, S3-хранилище (MinIO/AWS), SMTP для email-уведомлений. REST API с Swagger-документацией.

Интерфейс

Стек технологий
